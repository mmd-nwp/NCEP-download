# NCEP model download

NCEP model download is a software application to download NCEP model data to a local hard drive. 

---

## 🚀 Quick Start
perl model_download.pl

Command line options:
   -m model name [gfs, nam, hrrr, nbm] (default = gfs)
   -d init date (yyyymmddhh, default = latest)
   -n max fcst (default = model dependent)
   -p process "p" previous init time (default = 0)
